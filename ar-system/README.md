# AR Nav Vuforia Community

AR Navigation template with Unity and Vuforia.

The making of this project is documented as [weekly live streams on YouTube](https://www.youtube.com/playlist?list=PLcYgptwJHxUEM5xzPJJnWkXcKKfD7oRkO).

Repository for the backend (server): [indoor-navigation-backend](https://gitlab.com/drewjosh/indoor-navigation-backend)

Repository for the backend (web app): [indoor-navigation-admin](https://gitlab.com/drewjosh/indoor-navigation-admin)

Birth of this repo: 09.09.2024 🫀
